# Instalação de dependencias

API:
  - Acesse a pasta /API
  - Execute NPM install ou YARN para instalar as dependências
  - Rode a api com YARN DEV ou NPM DEV

FRONT: 
  - Acesse a pasta /Front
  - Execute NPM install ou YARN para instalar as dependências
  - Rode o projeto com YARN START ou NPM START
